# pwn_model
pwn模版，能够能加快速的致力于pwn的解题过程，简化pwntools代码量，快速写出pwn的解题代码
</br>
自动的识别32位和64位程序，可选择是否开启debug模式，只需要将pwn1改成自己的可执行文件即可，远程的话将pwn.challenge.ctf.show:28156改成对应的IP:PORT即可，攻击远程环境使用python3 python3_pwn_model.py REMOTE即可使用远程
